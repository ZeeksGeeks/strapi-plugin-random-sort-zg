# Strapi plugin random-sort

Simple plugin to randomly sort the data of a request.

## Installation

`npm install strapi-plugin-random-sort`

or

`yarn add strapi-plugin-random-sort`

## Config

```
  'random-sort': {
    enabled: true,
  },
```

## Usage

Add a `?randomSort=true` as a query parameter
